# Prueba tecnica Pokemon
Es una prueba tecnica para el puesto middler senior, la estructura esta hecha con React + Vite

|   |   └───images

|   |   └───Component

### Ambiente local

Para ejecutar la aplicación en su equipo de manera local, ejecute el comando:

* **Npm**:
```bash
npm run dev
```

* **Yarn**:
```bash
yarn run dev
```
